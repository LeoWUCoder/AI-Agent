## 数组查找功能
```C
#include <stdio.h>
// 实现数组查询功能
int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    int query;
    printf("请输入要查询的数字：");
    scanf("%d", &query);
    
    int found = 0; // 标志变量，表示是否找到
    for (int i = 0; i < 5; i++) {
        if (arr[i] == query) {
            printf("数字 %d 在数组中，索引为 %d\n", query, i);
            found = 1; // 找到后设置标志变量
            break; // 退出循环
        }
    }
    
    if (!found) {
        printf("数字 %d 不在数组中\n", query);
    }
    
    return 0;
}
```

C语言中没有办法直接定义字符串数据类型——所以用数组
```c
char arr[]="Leo";
```

### 常用字符串函数
strlen()获取字符串长度：1.不包括\0；2.汉字和字母不一样
strcmp(str1，str2)比较字符串的ASCII码，返回0表示一样，返回1表示str1的ASCII更大，返回-1表示str2的ASCII更大。str1和str2都是数组名
strcpy(str,"Sunny")表示覆盖原来字符串且不能对字符串常量进行拷贝，即原来的str[]储存的字符串变成了Sunny。注意：这里的str是字符串数组的名字，我们不能对字符串常量进行拷贝
strcat(s1,s2)用于拼接两个字符串，把s2的内容加到s1，所以意思是你要是像打印拼接后的结果，其实就是打印s1就好了。

# 指针
## 内存
我们知道计算机上CPU（中央处理器）在处理数据的时候，需要的数据是在内存中读取的，处理后的数据也会放回内存中，那我们买电脑的时候，电脑上是8GB/16GB/32GB等，是把内存划分为⼀个个的内存单元，每个内存单元的大小取1个字节。
1Byte（字节） = 8bit
1KB = 1024Byte
1MB = 1024KB
1GB = 1024MB
1TB = 1024GB
1PB = 1024TB
>“电脑 16GB 内存”“MCU 32KB 内存” → 都是 RAM。
RAM（内存） = 程序运行时使用，用来存放变量、堆栈、缓存等
掉电后数据会消失

在计算机中我们把内存单元（字节）的编号也称为地址。C语言中这个内存单元的编号，这个地址他就叫**指针**。
所以指针=地址，定义一个指针变量就是为了存储地址
每个地址可以存放一个字节的数据，
计算机内是有很多的硬件单元，⽽硬件单元是要互相协同⼯作的。所谓的协同，⾄少相互之间要能够进⾏数据传递。为了进行数据传递，引出 **“线”** 的概念。
![alt text](image.png)
| 特性 | 数据总线 (Data Bus) | 地址总线 (Address Bus) | 控制总线 (Control Bus) |
|------|---------------------|------------------------|------------------------|
| 功能 | 传输实际数据 | 传输内存/设备的地址信息 | 传输控制信号和时序信号 |
| 传输方向 | 双向（可读可写） | 单向（CPU→内存/设备） | 多为单向（部分双向） |
| 位数影响 | 决定每次传输的数据量 | 决定最大寻址空间 | 决定控制信号的丰富程度 |
| 典型宽度 | 8/16/32/64位 | 16/20/32/36位等 | 多条独立信号线 |
| 举例信号 | D0-D63 | A0-A35 | READ、WRITE、IRQ、RESET等 |

我们可以简单理解，32位机器有32根地址总线，每根线只有两态，表⽰0,1【电脉冲有⽆】，那么⼀根线，就能表⽰2种含义，2根线就能表⽰4种含义，依次类推。32根地址线，就能表⽰2^32种含义，每⼀种含义都代表⼀个地址。
**地址信息被下达给内存，在内存上，就可以找到该地址对应的数据，将数据在通过数据总线传⼊CPU内寄存器。**
## 指针变量
### 基础语法
指针变量其实就是代表着存储地址的变量，那么指针的类型其实取决于他存储的这个地址，指向的值的类型：类型名*指针变量名
```c
a='W';
char* pa;// 定义指针变量为pa
pa = &a；// pa作为指针变量只能存储地址
```
**pa是一个指针=pa是一个地址=这个地址指向a这个变量.**
&是取地址运算符
如果要访问指针变量的数据，要用取值运算符*。其实就是用这个地址，去查这个地址住的数据具体是啥。
```c
printf("%c",*pa);// %c 是用于输出 单个字符 的格式控制符。
```
### 避免未初始化的指针
如下格式：
```c
void f()
{
    int*a;
    *a=10;
}
```
这是一个非常危险的代码：它仅仅创建了一个“可以装地址的变量”。但它里面没有存任何有效地址。相当于无地址乱赋值。
